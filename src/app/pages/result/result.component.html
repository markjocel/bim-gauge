Recommendations

Coordination Each model element should be well coordinated across all discipline, perform thorough review before releasing information, and use the BIM model to directly generate information like plans, sections, and schedules.
BIM Use Use the BIM Model to create a construction sequence, quantity take-off for cost estimation and provide animations and rendered view or images.
Information Flow Use Common Data Environment (Single source of Truth) on your process. Please check ISO-19650 guidelines.
Working Practice Have a strategic workflow for BIM, regularly check clash detection on the model as agreed on the BEP, and have a standard for Level of Information Need.
System Have a program that will continously upgrade the Software and Hardware to support the latest Software on BIM.
IT Security Often create a data backup for security
Organizational Requirements Work on agreed BEP. Please refere ISO-19650-1 and 2.
Strategy Have a full management support to use BIM in the project.
Awareness and Motivation Create a BIM implementation strategy to fully implement BIM and reap off its benefits.
Information Exchange Provide Project wide file naming, version control and status compliant with recognised BIM Standard. Please refer to ISO-19650
Data Structure Create a company standard for Level of Geometry (LoG) and Level of Information (LoI)
Roles and Responsibilities Have a clear and defined BIM roles on the organization.
Qualification and Training Provide BIM training for employee.
Knowledge Management Provide a mechanism for staff to provide feedback on the information structure and BIM products.
